光的全反射

当光在介质界面上的入射角超过临界角时，
所有光线都会反射回原介质——这就是全反射。

🔍

发生条件

光从光密介质（折射率高）射向光疏介质（折射率低），且入射角 ≥ 临界角。

● n₁ > n₂ ● θ ≥ C
📐

临界角公式

sin C = n₁ / n₂

（n₂ > n₁，C 为临界角）

C 为临界角，n₂ 为入射侧（光密），n₁ 为折射侧（光疏）。当 θ = C 时折射光沿界面传播。

💎

光密 vs 光疏

光密介质指折射率较高的介质（光速更慢），如水(1.33)、玻璃(1.5)、钻石(2.42)。光疏介质指折射率较低的介质（光速更快），如空气(1.0)。全反射只能发生在光从光密→光疏的方向。

折射角的极限

随着入射角增大，折射角也增大。当入射角达到临界角 C 时，折射角恰好为 90°，折射光沿界面传播。继续增大入射角，则不再有折射光，所有能量反射回入射介质。

🎮 玩法说明

场景中有 激光发射器（红色）和 接收器（绿色），中间有几段 光纤内壁（蓝色线段）。

拖拽光纤内壁两端旋转角度，光线会实时跟随变化。

光线碰到光纤内壁时会计算入射角：大于临界角(42°)就全反射，否则光线穿透失败。

💡 物理提示

这不是普通镜子反射，而是模拟光在光纤内壁上的全反射传播。只有入射角（光线与镜面法线的夹角）大于 42°（玻璃临界角）时，才能发生全反射。

如果入射角太小，光就会折射穿过去——就像真实的全反射条件一样！
